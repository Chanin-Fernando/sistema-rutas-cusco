# Sistema de Gestión de Rutas Óptimas – Cusco

Proyecto académico para la asignatura Programación III (UNSAAC, 2026).

## Algoritmos implementados
1. Ordenación y Búsqueda → QuickSort, Búsqueda Binaria
2. Greedy → Vecino más cercano
3. Divide y Vencerás → Segmentación por cuadrantes
4. Programación Dinámica → Knapsack 0/1
5. Backtracking → Rutas con calles restringidas

## Requisitos
- Python 3.8 o superior
- Tkinter (incluido por defecto en Windows/Mac, en Linux instalar `python3-tk`)

## Ejecución
```bash
python main.py
